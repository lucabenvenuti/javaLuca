Package Specification

This package contains classes representing shapes. Base interface {@link Shape} defines methods for retrieving information of shapes, like position and size, and methods for drawing.

Concrete shape implementations are

  • Rect : representing rectangle with width and height.
  • Oval : representing an oval also specified with width and height.
  • Group : a group consisting of a list of sub-shapes of any type. Implements the composite pattern.

Class {@link PrimitiveShape} serves as a common base class for classes Rect and Oval.
