Package with the tool implementation for the graphical editor application. Tools implement the user interactions. Class Tool is an abstract base class for tools. Currently there are three concrete tool implementations:
  • RectTool for creating rectangles
  • OvalTool for creating ovals
  • SelTool for doing selections and moving and resizing shapes
A tool palette implemented in class ToolPalette is used for maintaining the tools in the application. In particular it has a currently selected tool.
